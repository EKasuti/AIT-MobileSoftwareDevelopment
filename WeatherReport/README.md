## Installation
1. Clone the repository:
   ```
   git clone https://github.com/EKasuti/AIT-MobileSoftwareDevelopment.git
   cd WeatherReport
   ```
2. Open the project in Android Studio
3. Build and run the application on an emulator or physical device
